# Remove or archive from the old plan

The following concepts are not supported by the Persian business document and should be removed from active planning unless independently required:

- generic BPMN workflow designer;
- Camunda or Flowable integration;
- generic workflow engine;
- dynamic form builder;
- generic form submissions platform;
- enterprise organization-chart engine;
- arbitrary position hierarchy;
- Jasper report server;
- generic report designer;
- generic dashboard designer;
- low-code platform features;
- arbitrary service tasks;
- microservice architecture;
- Redis as a default dependency;
- MinIO as a default dependency;
- broad notification systems not tied to a confirmed requirement;
- source-code recovery from minified bundles.

Preserve reusable authentication, Angular, NestJS, database, Docker, and test infrastructure.
