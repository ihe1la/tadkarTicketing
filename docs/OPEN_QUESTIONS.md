# Open questions

Do not block the first implementation phases on these unless required.

- Are ticket attachments required?
- Is customer registration self-service or administrator-approved?
- What exact working hours and holidays apply?
- Is password recovery required for customers?
- Are email or SMS notifications required?
- Which Active Directory groups map to each product/department?
- Is TFS on-premises or Azure DevOps Services?
- Should PBI creation be automatic or should the system only store an externally created PBI ID?
- Can managers reopen closed tickets?
- What are the final root-cause category names?
- What rating scale should customers use?
- What exact fields define geographic reporting?
- Is PDF export required, or is CSV sufficient?
